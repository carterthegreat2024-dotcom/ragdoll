# Ragdoll

## Installation

To install the Ragdoll application, please follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/carterthegreat2024-dotcom/ragdoll.git
   ```

2. Navigate into the project directory:
   ```bash
   cd ragdoll
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```

## System Architecture

The Ragdoll application is built on a modular architecture comprising the following components:

- **Frontend**: Handles user interactions and displays data.
- **Backend**: Manages application logic, database interactions, and APIs.
- **Database**: Stores application data, user information, and logs.

### Component Interaction

1. The Frontend communicates with the Backend through RESTful APIs.
2. The Backend interacts with the Database to retrieve and store data.
3. The Frontend renders the data received from the Backend for user display.
